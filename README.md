Bem-vindo! Esta pasta é um "arsenal" de modelos prontos de **prompts de engenharia** (instruções) para usar com Inteligências Artificiais Generativas (como Gemini, ChatGPT, Copilot, etc.).

O objetivo **não é** substituir o conhecimento do desenvolvedor, mas sim **acelerar o trabalho** e **mostrar as possibilidades** que a IA oferece.

A ideia é simples: **mostrar como podemos aprender facilmente a interagir com a IA** para resolver problemas reais do dia a dia no Mainframe ou qualquer outra linguagem ou ainda outros processos relacionados, seja uma revisão de código, uma criação de massa de dados baseado em uma estrutura sistemica do momento.
